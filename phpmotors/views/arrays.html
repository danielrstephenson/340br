PHP Arrays

An array is simply a variable that contains more than one value. Each value in an array is referred to as an "element". And, PHP keeps track of each value in the element using a "key" (sound familiar?)

Array Types

PHP has three types of arrays:

  1. Numeric (or indexed) array

    A numeric array uses numbers (starting with zero) as the keys for each element's value.

  2. Associative array

    An associative array uses strings as the keys for each element's value.

  3. Multi-dimensional array

    A multi-dimensional array is literally an array of arrays.
