jQuery Form Validation

Introduction

Validating the inputs from a form is critical to ensuring that the data you expect is the data you receive. When done in the browser it does not require data be sent to the server prior to being validated. The term Client-side refers to checking of data while in the browser.

Checking data in the browser is typically done using Native HTML5 Methods or using JavaScript. This document will discuss using JavaScript with the jQuery library.

JavaScript

To provide support across all browsers, JavaScript can be used. However, it too has a weakness. JavaScript can be disabled in the browser. When that occurs, the data is sent directly to the server. This is why identical checks of data must be done in both the browser and on the server!
