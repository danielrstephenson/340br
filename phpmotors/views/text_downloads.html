Download Text Materials

Download

  1. Download the zip file containing the chapter and exercise files for the text.
  2. Unzip the downloaded file.

Contents

The unzipped folder should be named "murach_files".

Open the folder. There should be 3 sub-folders:

  • book_apps — Contains the files and code found in the book chapters.
  • ex_starts — Contains the starting files for the end of chapter exercises.
  • ex_solutions — Contains the solutions to the end of chapter exercises.

Copy to htdocs

In setting up the tools for the course you should have an "htdocs" folder inside the XAMPP folder on your computer. This local environment is the perfect location to place files in order to do the chapter work, exercises and enhancements.

As you need you can open and work with these files to follow along with code examples shown in the Murach text. By placing the murach_text folder within the htdocs folder, all you have to do is open NetBeans, find the file to work with, open it and begin working.
