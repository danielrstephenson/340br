Install PHP Manual Search

NetBeans has a wide variety of plugins available to add to the software to give it additional functionality. Most of these can be added via the Plugins dialog box found in the NetBeans Tools menu.

However, from time to time some plugins will disappear from the normal plugin repository but can still be found and added manually. This is one of those situations.

Directions

  1. Find the plugin on the NetBeans.org site - http://plugins.netbeans.org/plugin/21086/php-manual-search
  2. Download the plugin by clicking the "Download" button
  3. Launch NetBeans
  4. Go to the "Tools" menu and select the "Plugins" option
  5. Click the "Downloaded" tab
  6. Click the "Add Plugins..." button in the top-left corner
  7. Search for the downloaded plugin, typically this will be in your downloads folder
    • At the time of this writing the name of the plugin is 1395791666_it-sal-phpmanual.nbm
  8. Select the file and click "Open"
  9. The plugin will appear in the left-side panel.
  10. Click the checkbox to the left of its name.
  11. Click the "Install" button in the bottom-left corner.
  12. Accept the license agreement and click the affirmative answers until the installation is complete.
  13. Close the Plugins dialog box.

Use

When the plugin is installed a new icon will appear in NetBeans, a blue oval with PHP inside. This button takes you directly to PHP.net and to the documentation of any PHP function that is currently selected by the cursor or the cursor is within.

The php.net documentation is the official documentation for all PHP code. By using this tool you can learn about PHP from the official source. It is very useful.
