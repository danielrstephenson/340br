NetBeans Localhost Tutorial

Prerequisites

This tutorial assumes you have:

  • Downloaded and installed the most recent version of NetBeans (PHP version).
  • Completed the XAMPP installation and configuration of a local development environment on your own machine.
  • Cleaned up the HTDOCS folder inside of XAMPP on your local machine.

Video Version

Create a Domain Project

A project is a web site for the purposes of this tutorial. It is where all files and folders reside that pertain to a web site. NetBeans then uses its functionality to keep track of those resources and assists you to manage the site as a whole.

This tutorial takes you through the process of creating a project for your classwork on your local machine.

  1. Launch NetBeans
  2. Click the "Create New Project" icon (top left corner of the screen) or from the File menu choose "New Project"
  3. A new project dialog box will open showing PHP options (due to our using the PHP version of the tool)
  4. Click the second option â€” PHP Application with Existing Sources, click "Next"

Name and Specify the Project Location

  1. Type the name "Localhost" in the Project Name text box.
  2. Click the "Browse" button and navigate to the "htdocs" folder (on Mac: [Applications > XAMPP > htdocs] or on PC: [C:\XAMPP > htdocs]). Click to select it and then click "Open".
  3. Important! Make sure the Sources Folder path ends with "htdocs". If any other text appears after "htdocs" delete it!
  4. The PHP Version should be changed to the highest value available (e.g. 7.0).
  5. The Default Encoding settings can remain as they are. Click "Next".

Set the Run Configuration

The run configuration tells NetBeans what to do when a file is to be previewed or "run".

  1. By default, the Run As: box is set to "Local Web Site (running on local web server)". Do not change this.
  2. In the "Project URL" text box, make sure that the path is http://localhost.
  3. The "Index File:" should be index.php.
  4. Click "Finish" to save the settings and close the configuration window.

The Project Panel

When you return to NetBeans, the Project panel should be viewed on the left side of your screen. If it is not visible, click the "Window" toolbar and select "Project". Insure that you have a project named "Localhost". This is where you will develop your web site files for testing on the local machine.
