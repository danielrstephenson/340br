Final Project

Description

The final project is the major project of the semester. The project must be built to the specifications provided to you in week 12. The following basics will be included:

  • Use of an M-V-C architecture (frameworks created by others are not allowed).
  • Views must be use semantic markup, responsive to large through small device screens and be valid HTML5 and CSS3;
  • Use custom functions to facilitate common code processes;
  • Use all four CRUD data processes involving the database;
  • Incorporate server and client-side data validation to protect the database and insure correct data submission;
  • Use the PDO library and prepared statements for database interactions within the model;
  • Build and use a database and/or database tables to match provided specifications.

Objectives and Values

  • All course objectives will be assessed in the project.
  • Each objective is worth 25 points.

Timeline

  1. Project document will be available at the beginning of week 12.
  2. Project will be due the last published day of classes for the semester at 12pm (Noon) Mountain Time - consult the course calendar.

Grading Matrix

A grading matrix will be provided with the project requirements in week 12.

Peer Reviews

Two peer reviews of your project will be required. Consult the peer review documentation for details.
